# PKU Course Planner (26fall)

一个用于浏览课程信息和模拟排课的网页工具。支持课程搜索、筛选、加入课表、时间冲突检测和周课表导出。基于北京大学26-27学年第1学期课表数据，来源于北京大学教务部全校课表查询 (https://dean.pku.edu.cn/service/web/courseSearch.php)。

本工具仅用于模拟选课和排课规划，不会连接学校选课系统，也不会提交真实选课结果。

## 功能

### 课程搜索

支持按以下内容搜索课程：

- 课程名称
- 课程号
- 教师
- 开课单位
- 课程类型
- 备注

### 课程筛选

支持通过左侧筛选栏筛选课程：

- 按开课单位筛选
- 按课程类型筛选
- 只显示有上课时间的课程

### 分页浏览

搜索结果支持分页查看，避免一次显示过多课程。

### 加入我的课表

可以将感兴趣的课程加入“我的课表”。

已加入的课程会显示为“已加入”。

### 时间冲突检测

添加课程时，系统会自动检测该课程是否与已选课程存在时间冲突。

冲突检测主要基于：

- 星期
- 节次
- 起止周
- 单双周

如果存在冲突，系统会提示冲突课程和冲突时段。

### 允许添加冲突课程

如果确实想保留时间冲突的课程，可以在左侧勾选：

```text
允许添加冲突课程
```

勾选后，即使课程冲突，也可以继续加入课表。

### 我的课表

“我的课表”页面会自动生成周一到周日、第 1–13 节的模拟课表。

课表格子中显示：

- 课程名称
- 教师

课表列宽会根据课程内容自动调整。

### 删除课程

可以从“我的课表”中删除已选课程。

### 清空课表

可以一键清空当前已选课程。

### 导出周课表

支持下载当前周课表 CSV 文件。


## 使用方法

### 1. 搜索课程

在左侧筛选栏的关键词框中输入关键词。

例如：

```text
心理
数据结构
通选课
数学科学学院
```

搜索范围包括课程名称、课程号、教师、开课单位、课程类型和备注。

### 2. 筛选课程

可以通过左侧下拉菜单选择：

- 开课单位
- 课程类型

如果只想查看有固定上课时间的课程，可以勾选：

```text
只看有上课时间的课程
```

### 3. 翻页查看课程

搜索结果较多时，可以使用页面底部的分页按钮：

```text
上一页
下一页
```

查看不同页的课程结果。

### 4. 加入课表

在课程卡片中点击：

```text
＋ 加入课表
```

课程会被加入“我的课表”。

如果该课程与已选课程存在时间冲突，系统会弹出冲突提示。

### 5. 查看我的课表

点击页面上方的：

```text
我的课表
```

即可查看已选课程和自动生成的周课表。

### 6. 删除已选课程

在“我的课表”页面中，点击课程卡片下方的：

```text
删除
```

即可将该课程从课表中移除。

### 7. 清空我的课表

点击左侧的：

```text
清空我的课表
```

可以清除当前已选的所有课程。

### 8. 下载周课表

在“我的课表”页面中点击：

```text
下载周课表 CSV
```

即可下载当前模拟课表。

CSV 文件会按照以下结构导出：

```text
节次,周一,周二,周三,周四,周五,周六,周日
第1节,...
第2节,...
...
第13节,...
```

## 注意事项

- 本工具只是模拟排课工具，不代表真实选课结果。
- 本工具不会连接学校选课系统，也不会提交真实选课。
- 课程信息请以学校教务系统发布内容为准。
- 时间冲突检测主要基于课程表中的标准上课时间。
- 对于调课、补课、考试安排、特殊备注等复杂信息，本工具只能做基础处理。
- 已选课程保存在当前浏览器中，更换浏览器或清除浏览器缓存后可能会丢失。
